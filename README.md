🚀 Smart Object Interaction Dashboard

A futuristic AI-powered web application that uses TensorFlow.js COCO-SSD and Webcam to detect real-world objects and trigger smart UI interactions in real time.

This project runs fully on localhost and provides a dark, glassmorphism-based futuristic dashboard.

✨ Features
🎥 Real-Time Object Detection
Live webcam feed
COCO-SSD TensorFlow.js model
Real-time object detection
Multiple object detection
Top 3 object highlighting
Confidence score display
Neon bounding boxes
🧠 Smart Object Interactions
Object	Action
👤 Person	User Present Indicator

💻 Laptop	Coding Mode Panel

📦 Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/yourusername/smart-object-dashboard.git
2️⃣ Navigate to Project Folder
cd smart-object-dashboard
3️⃣ Run Using Live Server 

If using VS Code:

Install Live Server Extension

Then right-click:

index.html → Open with Live Server
Alternative Method (Python Server)
python -m http.server 5500

Then open:

http://localhost:5500
🚀 How It Works
App asks for webcam permission
Loads TensorFlow.js COCO-SSD model
Starts real-time object detection
Detects supported objects
Triggers UI interactions
